# Viajes Chile

Landing page desarrollada para la prueba **Viajes Chile**.

## Recursos usados

- HTML5 semántico
- CSS3
- Bootstrap 5.3.3
- Bootstrap JS: carousel, tooltips y popover
- Google Fonts: Raleway
- Font Awesome 6.5.2
